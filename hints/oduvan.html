Como contar a frequência usando um if dentro de um loop?

Você pode criar uma lista vazia para armazenar as frequências e usar um loop for com um if para contar manualmente a frequência de cada elemento.

Deve ter um jeito mais fácil...

Você pode usar a função count para contar quantas vezes cada elemento aparece na lista. Use um for para contar todos os elementos da lista. 

hashtags = ["#python", "#java", "#python", "#javascript"]
hashtags.count(#python))

Quero deixar meu código mais eficiente, pois estou contando todos os elementos inclusive os que se repetem.

Você pode remover os itens já contados cada vez que eles aparecem. Você pode usar o método .pop() ou o método .remove(). Nesse caso é mais fácil usar o remove() pois o parâmetro fornecido é o valor da palavra, já o .pop() remove por índice. 

lista = [10, 20, 30, 40, 50]
lista.remove(30)  # Remove a primeira ocorrência de 30
print(lista)  # Saída: [10, 20, 40, 50]

Tem outro jeito?

Você pode usar a função set() para criar uma coleção dos elementos. lista = [1, 2, 2, 3, 4, 4, 5] conjunto = set(lista) print(conjunto) # Saída: {1, 2, 3, 4, 5} Mas atenção! O os elementos do conjunto nem sempre aparecem na mesma ordem da lista.
